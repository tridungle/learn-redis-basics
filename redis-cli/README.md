# redis-cli

## On MacOS (installed using Homebrew)

1. Start redis-cli

    ```shell
    redis-cli
    ```

## On Ubuntu

1. Start redis-cli

    a. Navigate to redis folder and run
    ```shell
    src/redis-cli
    ```
